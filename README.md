# 감정들(Emotions)

'감정들'은 현재의 감정을 발견하고, 표현하고, 공유하고 싶은 사람들을 위해<br>
감정카드를 선택하고 감정일기를 적어 나누는 익명 기반 감정 커뮤니티 앱 입니다.<br>
감정을 적고, 댓글을 달며 서로의 감정을 나누어 보세요.<br>
따뜻한 위로와 응원이 여러분을 기다립니다.
